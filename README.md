# 🚀 ATHENA MD

A powerful, modern, and fully-featured WhatsApp bot built for automation, entertainment, and advanced control.

ATHENA MD is designed to be fast, stable, and easy to manage — whether you're running a community, testing automation, or building your own system.

---

## ✨ Features

### 🤖 Core System
- Fast and stable WhatsApp bot (Baileys-based)
- Clean command handling system
- Modular structure (easy to edit and extend)
- Auto-update system via GitHub

---

### ⚙️ Admin Tools
- Kick, ban, unban users
- Promote / demote members
- Mute / unmute groups
- Tag all / hide tag
- Anti-link system
- Anti-badword protection

---

### 📢 Channel Integration
- Newsletter (WhatsApp Channel) support
- Auto channel preview in messages
- Easy channel linking via global config

---

### 🎮 Fun & Games
- Trivia
- Hangman
- TicTacToe
- Random jokes, facts, quotes
- Truth or Dare commands

---

### 🎵 Media & Download
- YouTube audio/video download
- TikTok downloader
- Spotify support
- Image tools (blur, stickers, etc.)

---

### 🧠 AI Features
- AI chat commands
- Smart responses
- Extendable AI system

---

### 🔒 Security & Control
- Owner / sudo system
- PM blocker
- Anti-delete (message recovery)
- Anti-call protection

---

---

## ⚙️ Requirements

Before running the bot, ensure you have:

- Node.js (v18 or higher recommended)
- Stable internet connection
- A panel (Pterodactyl recommended) or VPS
- WhatsApp account
